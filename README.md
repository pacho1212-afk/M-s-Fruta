# Más Fruta 🍓

Sitio web de Más Fruta — heladería y frutería con sedes en Pasto, Ipiales y Santander de Quilichao.

Construido con HTML5, CSS3 y JavaScript puro (sin frameworks).

## Publicar en GitHub Pages (gratis)

1. Crea un repositorio nuevo en GitHub, por ejemplo `masfruta`.
2. Sube todo el contenido de esta carpeta a la raíz del repositorio (no dentro de una subcarpeta).
3. En el repositorio ve a **Settings → Pages**.
4. En "Build and deployment", selecciona **Deploy from a branch**.
5. Elige la rama `main` y la carpeta `/ (root)`, luego **Save**.
6. Espera 1–2 minutos. Tu sitio quedará publicado en:
   `https://TU-USUARIO.github.io/masfruta/`

## Notas

- Las imágenes de productos y galería son temporales de Unsplash. Reemplázalas en `index.html` por fotos propias cuando estén listas.
- Actualiza los números de WhatsApp y direcciones reales en la sección de Sedes y en el footer.
- `manifest.json` deja el sitio listo para instalarse como app (PWA); si quieres los íconos PNG reales (192x192 y 512x512) dímelo y los genero.
